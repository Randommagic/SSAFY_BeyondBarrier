## 📌 프로젝트 소개

### Beyond Barrier는 AI를 활용한 TV 화면 해설 및 프로그램 정보 제공 서비스입니다

#### 사용자
  -  시각 장애인 및 저시력자 분들을 타겟으로 했습니다

#### 주요 기능
  -  시각 장애인, 저시력자 분들의 모바일 앱 사용을 위한 튜토리얼을 제공합니다
  -  현재 장면에 대한 해설을 음성으로 제공합니다
  -  방영 중인 방송 프로그램의 이름과 설명, 주요 출연진을 음성으로 알려줍니다
  -  안내 음성 속도와 해설 언어 등을 설정할 수 있습니다

#### 장점
  - 방송되는 장면의 내용과 프로그램 정보를 제공하여 시각적으로 어려움을 가진 분들께 보다 풍부한 경험을 제공합니다
  - 기존에 사람이 직접 해설하던 방식과는 달리 AI를 도입해 장애인 방송 제작 비용을 줄일 수 있습니다
  - 지상파 방송 뿐 아니라 화면 해설 서비스가 미흡한 VOD나 OTT 시장에도 장애인 접근성 컨텐츠를 제공할 수 있습니다
